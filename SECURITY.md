# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in monetizeyouragent.fun (API, MCP server, payment logic, or website), please report it responsibly.

**Email:** security@monetizeyouragent.fun

**Do NOT** open a public GitHub issue for security vulnerabilities.

## Scope

- REST API endpoints (`/api/v1/*`)
- MCP server (`/mcp`)
- Tweet-to-Earn payment logic
- Rate limiting and input validation
- Authentication and admin panel

## Response

We aim to acknowledge reports within 48 hours and provide a fix or mitigation plan within 7 days for critical issues.
